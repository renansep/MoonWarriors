MoonWarriors 战神传说
==================

This showcase utilizes many features from "Cocos2d-html5 engine",including:
   * Parallax background
   * tilemap
   * actions
   * ease
   * frame animation
   * schedule
   * Labels
   * keyboard Dispatcher
   * Scene Transition
